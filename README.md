# Студенческий журнал

Легкий проект студенческого журнала написанного на Spring c элементами HTML.
В проекте можно создавать студента, изменять и удалять.

Использовал технологии:
<li>Spring Boot
<li>Spring MVC
<li>Spring Data JPA ( Hibernate)
<li>MySQL
<li>Thymeleaf
